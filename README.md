# 📚 TaskSchool - Plataforma de Gestión de Tareas

## 📌 Descripción
TaskSchool es una aplicación web diseñada para ayudar a los estudiantes a gestionar sus tareas académicas de manera eficiente. Permite crear, editar, eliminar y consultar actividades.

## 🎯 Objetivo
Facilitar la organización académica mediante una plataforma digital accesible y fácil de usar.

## 🛠 Tecnologías utilizadas
- Frontend: HTML, CSS, Bootstrap
- Backend: Node.js, Express
- Base de datos: MySQL
- Herramientas: Git, GitHub, Visual Studio Code

## 🧩 Arquitectura del sistema
El proyecto sigue una arquitectura cliente-servidor:

- Frontend: Interfaz del usuario
- Backend: Procesamiento de datos
- Base de datos: Almacenamiento de información

Flujo:
Usuario → Web → Servidor → Base de datos → Respuesta

## 📂 Estructura del proyecto
- frontend → interfaz gráfica
- backend → lógica del servidor
- database → base de datos
- docs → documentación

## 🚀 Funcionalidades
- Registro de usuarios
- Inicio de sesión
- Crear tareas
- Ver tareas
- Editar tareas
- Eliminar tareas

## 📸 Evidencias
(AQUÍ METES IMÁGENES)
- Login
- Dashboard
- Crear tarea

## ⚙️ Instalación
1. Clonar repositorio:
git clone (tu link)

2. Instalar dependencias:
npm install

3. Ejecutar servidor:
node server.js
